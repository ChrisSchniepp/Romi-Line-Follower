Detailed Description

This file contains the implementation of the QTRX class, which provides functionality for interacting with a QTRX reflectance sensor array for line following and navigation tasks.

The QTRX class enables reading raw sensor values, normalizing them, and detecting a line based on the reflectance characteristics of a surface. It supports centroid calculation for precise line positioning.

  • The class initializes with GPIO pins for each sensor and an optional pin for controlling IR LEDs.
  • Supports individual and batch sensor readings, including decay time measurement for reflectance detection.
  • Provides methods to normalize sensor data and apply thresholding for binary line detection.
  • Includes a centroid calculation method to determine the position of a detected line relative to the sensor array.
  • Suitable for integration in real-time robotics systems for tasks such as line following.
