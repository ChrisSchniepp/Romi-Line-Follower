Detailed Description

This file contains the implementation of the IMU class, which provides functionality for interacting with the BNO055 inertial measurement unit (IMU) sensor using the I2C protocol.

The IMU class enables the retrieval and processing of orientation and angular velocity data, as well as calibration of the BNO055 sensor. It supports multiple operational modes and facilitates real-time feedback for robotics and motion control applications.

  • The class initializes with an I2C controller for communication with the BNO055 IMU.
  • Supports mode switching among NDOF, IMU, Compass, and more for versatile usage scenarios.
  • Includes methods to retrieve calibration status, read Euler angles, angular velocities, and yaw rates.
  • Provides functionality to load and save calibration data to/from binary files for persistent calibration.
  • Includes auto-calibration routines for ensuring accurate sensor feedback.
  • Designed for integration in robotic systems requiring precise orientation feedback.
