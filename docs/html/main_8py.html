Detailed Description

This file implements the main control logic for a Romi Robot, integrating motor control, encoder feedback, IMU data, and QTRX sensor readings to perform tasks such as line following, navigation, and obstacle handling.

The main.py script coordinates multiple subsystems of the Romi Robot, utilizing cooperative multitasking for efficient real-time control. It incorporates inner and outer control loops to manage motor speeds and robot orientation while responding to sensor inputs.

  • Subsystems Integrated:
    • Motor control with PID controllers for precise motion.
    • Encoder feedback for velocity measurement and position tracking.
    • IMU for yaw rate and heading measurements.
    • QTRX sensor array for line following and centroid detection.
    • Bumper inputs for obstacle handling and interaction.
  • Task Scheduling:
    • Inner loop task: Runs at a higher frequency for motor speed control.
    • Outer loop task: Runs at a lower frequency for velocity and yaw rate control.
  • Controllers Used:
    • PID controllers for longitudinal, yaw, and line-following control.
    • Decoupling matrix for motor speed calculation from velocity and yaw rate setpoints.
  • Features:
    • Dynamic line following with real-time QTRX feedback.
    • IMU-based orientation control with heading stabilization.
    • Bumper-triggered round block maneuvering.
    • Calibration routines for IMU initialization.
  • Designed for high-performance robotics applications on the STM32 platform.
